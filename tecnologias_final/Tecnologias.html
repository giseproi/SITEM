Cuando se desea elaborar una red de Telemedicina es preciso tener en cuenta una serie de factores que son de gran importancia para su correcto funcionamiento. Uno de estos elementos es la tecnología utilizada para interconectar la red, por este motivo se ha construido un espacio en donde se pueda contar con esta información.

Con el fin de brindar datos específicos de cada uno de las tecnologías de interconexión para servir de soporte a entidades que deseen implementar redes de Telemedicina, se cuenta con esta herramienta que administra y muestra información acerca de las Tecnologías de interconexión útiles en una red Telemédica.
